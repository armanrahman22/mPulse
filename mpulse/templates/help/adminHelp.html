Overview

In addition to providing end-users with access to their M-Pulse session data, this website also provides M-Pulse kiosk developers and administrators to some remote monitoring tools for their kiosks. In order to use these tools and to store data from your kiosk online, you must first register each of your kiosks with us. Upon registration, you will receive a registration key that you can enter either into the kiosk software install script or into the registerKiosk.sh script. For more information on registration and installation, see the help section Kiosk Hardware -> Remote Management

The remote management tools available to developers and administrators are:

  • Ability to view the status of kiosks (online, offline, error) and their IP addresses
  • Ability to view uploaded kiosk logs
  • Control who can view and manage your kiosks
Registering your kiosks also allows users to:
  • Store session data online
  • Email session data to themselves
  • Graph session data over time

Register Kiosk

Kiosks can be easily registered on the register kiosk page.

Manage Kiosks

You can view a list of kiosks you manage and their statuses on the kiosk admin page. This is also where you can view kiosk logs and edit kiosks.

Edit Kiosk

By clicking 'Edit Kiosk' on the kiosk admin page, you can change kiosk parameters and control who has ownership of the kiosk. Kiosk owners can view and edit the kiosk on their admin page.
